* [Angular 6](https://www.techiediaries.com/angular-tutorial/)
* Basic knowledge of [GIT](https://www.youtube.com/watch?v=cEGIFZDyszA&list=PL6gx4Cwl9DGAKWClAD_iKpNC0bGHxGhcx)
